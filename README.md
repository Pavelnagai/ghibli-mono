# Ghibli Project

Монорепозиторий для проекта Ghibli, содержащий фронтенд и бэкенд приложения.

## Структура проекта

```
ghibli/
├── apps/                    # Приложения
│   ├── front/              # Фронтенд приложение
│   └── backend/            # Бэкенд приложение
├── packages/               # Общие пакеты
│   ├── shared/            # Общий код
│   └── ui/                # UI компоненты
├── tools/                  # Инструменты и скрипты
├── docs/                   # Документация
```

## Требования

- Node.js (v18 или выше)
- npm (v8 или выше)

## Установка

1. Установка зависимостей:
```bash
npm install
```

2. Запуск в режиме разработки:
```bash
npm run dev
```

## Доступные команды

- `npm run dev` - Запуск всех приложений в режиме разработки
- `npm run build` - Сборка всех приложений
- `npm run test` - Запуск тестов
- `npm run lint` - Проверка кода линтером
- `npm run clean` - Очистка кэша и node_modules
- `npm run format` - Форматирование кода

## Разработка

### Добавление нового пакета

1. Создайте новую директорию в `packages/`
2. Инициализируйте package.json
3. Добавьте необходимые зависимости
4. Обновите tsconfig.json если нужно

### Добавление нового приложения

1. Создайте новую директорию в `apps/`
2. Инициализируйте приложение
3. Добавьте необходимые зависимости
4. Обновите конфигурацию в turbo.json если нужно
